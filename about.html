About

I'm Eddison Patrimonio, a 22-year-old Front-end Developer and Graphic Designer based in New Pandan, Panabo City, Philippines. I build clean, practical web interfaces — the kind made to solve one specific problem well, not just look good in a portfolio.

My main stack is HTML, CSS, Tailwind CSS, and JavaScript, with PHP for back-end logic and jQuery for interactive details. On the design side, I work on visual assets, layouts, and UI components that feel intentional and consistent.

Most of what I've built has come out of coursework: an inventory system for a convenience store, a recipe-sharing platform with an admin dashboard, and a vulnerability assessment tool that pulls together several open-source security scanners into one dashboard.
